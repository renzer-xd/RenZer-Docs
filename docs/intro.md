# RenZer Docs

ยินดีต้อนรับสู่คลังเก็บข้อมูล บันทึกความรู้ และการเรียนรู้ส่วนตัวของ **RenZer**

พื้นที่นี้สร้างขึ้นสำหรับบันทึกและจัดเก็บข้อมูล เทคนิค คู่มือการแก้ปัญหา และสิ่งต่างๆ ที่ได้เรียนรู้มา เพื่อใช้เป็นแหล่งอ้างอิงและทบทวนความรู้ได้ทุกเมื่อเมื่อเริ่มหลงลืม

## เริ่มต้นเขียนเอกสาร

คุณสามารถเริ่มต้นสร้างหมวดหมู่และเขียนบันทึกความรู้ของคุณเองได้โดยการ:

1. สร้างไฟล์ Markdown (`.md` หรือ `.mdx`) ในโฟลเดอร์ `docs/`
2. จัดหมวดหมู่ด้วยการสร้างโฟลเดอร์ย่อย และสร้างไฟล์ `_category_.json` เพื่อกำหนดชื่อกลุ่มเอกสาร
3. ตัวบอร์ดเอกสารจะอัปเดตเมนูนำทาง (Sidebar) ทางด้านซ้ายให้โดยอัตโนมัติ
